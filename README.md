# AngularJS ES6 Sample

> Flight Finder with AngularJS 1.6+ and EcmaScript 2015

## Run Steps

* Dependencies Install

```npm install```

* Run Test Cases

```npm test```

* Server Start

```npm start```

## Dependencies

* Angular JS 1.6+
  * Babel
  * ES6
* Webpack
* **Testing**
  * Karma: Test Runner
  * Chai: BDD asserts library
  * Mocha: Test Framwork
